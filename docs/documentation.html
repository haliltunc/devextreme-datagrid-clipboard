Documentation

Complete documentation for the DevExtreme DataGrid Clipboard plugin.

Installation

Install the plugin using npm:

npm install devextreme-datagrid-clipboard

Or include it directly in your HTML:

<script src="path/to/dx-datagrid-clipboard.js"></script>

Basic Usage

Initialize the plugin with a DataGrid instance:

const grid = $("#gridContainer").dxDataGrid({ // grid configuration }).dxDataGrid("instance"); const clipboard = new DataGridClipboard(grid, { showOperationInfo: true });

Methods

copy(options)

Copy selected or all rows to clipboard.

clipboard.copy(options?: CopyOptions): void

Options:

  • copyMode: "display" | "data"
  • includeHeaders: boolean
  • copyAllRows: boolean
  • includeRowNumbers: boolean
  • rowNumberColumnText: string

copyRows(rowIndexes)

Copy specific rows by their indexes.

clipboard.copyRows(rowIndexes: number[]): void

copyRowsByKey(keys)

Copy specific rows by their key values.

clipboard.copyRowsByKey(keys: any[]): void

paste()

Paste data from clipboard.

clipboard.paste(): void

setLanguage(lang)

Change the language for notifications.

clipboard.setLanguage(lang: string): void

setMessages(messages)

Set custom messages for notifications.

clipboard.setMessages(messages: object): void

destroy()

Destroy the clipboard instance and clean up resources.

clipboard.destroy(): void

Events

Event Parameters Description
onCopy data: string Fired when data is copied to clipboard
onPaste data: string Fired when data is pasted from clipboard
onError error: Error Fired when an error occurs during copy/paste

Browser Support

The plugin supports all modern browsers that implement the Clipboard API:

  • Chrome 42+
  • Firefox 41+
  • Safari 10+
  • Edge 12+
  • Opera 29+
